# Исправление проблемы с кодировкой

## Проблема
Русские тексты отображаются в неправильной кодировке: `РЈРїСЂР°РІР»РµРЅРёРµ РѕР±СЉРµРєС‚Р°РјРё РЅРµРґРІРёР¶РёРјРѕСЃС‚Рё`

Это происходит из-за неправильной интерпретации UTF-8 как Windows-1251 (или наоборот).

## Решение

В AggreGate описание контекста нельзя изменить после создания. Нужно:
1. Удалить контексты с неправильной кодировкой
2. Пересоздать их с правильной кодировкой

## План исправления

1. Удалить все созданные контексты
2. Пересоздать их с правильными описаниями
3. Проверить переменные и другие данные
