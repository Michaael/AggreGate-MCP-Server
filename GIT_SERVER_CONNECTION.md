
### Настройка git (если нужно)

## ⚠️ Безопасность

**ВНИМАНИЕ**: Пароль указан в открытом виде. Рекомендуется:
1. Использовать SSH ключи вместо пароля
2. Не хранить пароли в открытом виде в файлах
3. Использовать переменные окружения для чувствительных данных
