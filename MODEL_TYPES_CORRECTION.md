## Правильные значения типов моделей:

1. **Относительная модель (Relative Model)**: `type = 0`
   - Много экземпляров (по одному на объект)
   - Относительные ссылки в привязках: `{.:var}`
   - Требует `containerType` и `objectType`

2. **Абсолютная модель (Absolute Model)**: `type = 1`
   - Один экземпляр
   - Абсолютные пути в привязках: `{context:var}`

3. **Экземплярная модель (Instance Model)**: `type = 2`
   - Создается по требованию
